# Basic Backend Project

Blog App


### Installing

```
npm install
```
### Deploying (Dev)

```
prisma deploy -e ../config/dev.env
```
### Deploying (Prod)

```
prisma deploy -e ../config/prod.env
```
### Schema Generating

```
npm run get-schema
```

### generate token

```
prisma token
```

### DB reset

```
prisma delete
```
